# Информационная система театра

Десктопное клиентское приложение на **JavaFX** для работы с базой данных театра: просмотр и анализ данных, гибкие запросы с фильтрами, изменение данных с учётом прав доступа.

## Возможности

- Вход под одной из **5 ролей**: отдел кадров, постановщик, администратор репертуара, кассир, директор.
- Вкладка **«Запросы»** — аналитические выборки из каталога SQL (работники, репертуар, спектакли, билеты и др.) с параметрами и переключением **типа результата** (список или количество).
- Вкладка **«Просмотр»** — просмотр таблиц, доступных роли.
- Вкладка **«Изменение данных»** — добавление, правка и удаление записей в разрешённых таблицах; процедуры найма работника и назначения показа.

## Что должно быть на компьютере

| Компонент | Версия |
|-----------|--------|
| **JDK** | 17 или новее |
| **Apache Maven** | 3.9+ |
| **PostgreSQL** | 14+ (локально или в Docker) |
| **ОС** | Windows, Linux или macOS с графической средой (для окна JavaFX) |

Дополнительно для Docker: установленные **Docker** и **Docker Compose**.


## Подготовка базы данных

1. Запустите PostgreSQL. Проще всего через Docker:

   ```bash
   docker compose up -d
   ```

   Будет создана БД `theater_db` (пользователь и пароль: `postgres` / `postgres`, порт `5432`).

2. Создайте схему и заполните тестовыми данными:

   ```bash
   psql -h localhost -U postgres -d theater_db -f src/main/resources/sql/create_tables.sql
   psql -h localhost -U postgres -d theater_db -f src/main/resources/sql/insert_tables.sql
   ```

## Запуск приложения

```bash
mvn clean javafx:run
```

Сборка без запуска UI:

```bash
mvn clean compile
```

## Подключение к БД

Параметры заданы в `MainApp.java`:

| Параметр | Значение по умолчанию |
|----------|------------------------|
| Хост | `localhost` |
| Порт | `5432` |
| База | `theater_db` |
| Пользователь | `postgres` |
| Пароль | `postgres` |

## Структура проекта

```
src/main/java/com/theater/app/   — код JavaFX-приложения
src/main/resources/sql/          — SQL для приложения (схема, данные, запросы)
```
